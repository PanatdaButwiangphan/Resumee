การเขียนสะท้อนตนเองแบบการเรียนรู้ผ่านประสบการณ์Week # 1
(Recursive) ประสบการณ์ในสัปดาห์นี้มีการเชื่อมต่อกับประสบการณ์ที่ผ่านมาหรือไม่ ? (มี เคยฝึกงานใช้ html css ในการแก้เว็บให้ลูกค้า)
(CE) อะไรคือประสบการณ์ที่ผ่านมาของคุณ ? (ประสบการณ์ในการฝึกงาน ที่ได้ใช้งานจริง ทำงานจริง ทั้งในเรื่อง frontend backcend ซึ่งได้ใช้ทั้ง html css java javascript)์
(RO) อะไรคือประสบการณ์ที่สำเร็จและล้มเหลว ? (คิดว่าการฝึกงานผ่านและได้รับคำชมคือการประสบความสำเร็จอย่างหนึ่ง แต่สิ่งที่ล้มเหลวคือ เราไม่สามารถนำสิ่งที่ได้ฝึกงานมาใช้เต็มที่เพราะยังมีความรู้แค่เฉพาะทางที่ฝึกงานมา)
(AC) คุณได้เรียนรู้อะไรจากประสบการณ์ ? (ได้รู้ว่า เรื่องที่เราเรียนกันอยู่ทุกวันนั้นกว้างขวางมาก ต่อให้ศึกษาไปเท่าไหร่ยังไงก็จะมีภาษาใหม่ขึ้นมาให้เราได้เรียนรู้เสมอ )์
(AE) คุณจะได้นำสิ่งที่เรียนรู้จากประสบการณ์ไปใช้อย่างไรต่อไป ? (นำไปพัฒนา ให้เห็นถึงความล้มเหลวในอดีตเพื่อให้วันนี้เรากระตือรือร้นที่จะทำมันให้ดีที่สุดจะได้ไม่นึกเสียดายทีหลัง)
